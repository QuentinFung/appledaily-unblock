蘋果日報免登記會員 (Chrome Extension)
==========
[![Software License](https://img.shields.io/badge/license-MIT-brightgreen.svg)](LICENSE)

簡介
----
免登記成為會員都可安心睇蘋果日報🍎

安裝
----
1. 去 https://chrome.google.com/webstore/detail/, 下載並加到Chrome
2. 去蘋果日報
3. 唔使登記成為會員都可以睇新聞
